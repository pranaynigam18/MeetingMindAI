# MeetingMind AI

## Vision

MeetingMind AI is NOT an AI Meeting Summarizer.

MeetingMind AI is an Organizational Intelligence Platform.

It transforms meetings into long-term company memory.

The platform helps organizations:

- Preserve knowledge
- Track decisions
- Track action items
- Detect risks
- Detect repeated discussions
- Answer company-wide questions using AI

---

## Mission

Every feature must satisfy at least one objective:

- Save time
- Improve decision making
- Increase accountability
- Preserve organizational knowledge
- Detect risks
- Reduce repeated discussions

If a feature does not achieve one of these goals, it should not be built.

---

## Current Features

- Meeting Upload
- Whisper Transcription
- Gemini Analysis
- PostgreSQL Storage
- Dashboard
- Meeting History
- AI Search
- Decision Center
